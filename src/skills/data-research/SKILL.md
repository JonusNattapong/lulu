# Data Research

Conduct structured research and data extraction.

## When to Trigger
- User wants research on a topic
- Need to extract structured data
- Survey a topic thoroughly

## Steps
1. **Define scope** - Understand research question
2. **Search sources** - Use web search and brain
3. **Extract data** - Pull structured information
4. **Synthesize** - Combine into coherent answer
5. **Cite** - Include sources

## Quality Bar
- Structured output
- Multiple sources cited
- Confidence levels indicated
- Gaps acknowledged

## Tools Required
- brain_query
- web-search
- data extraction tools
