# CLI Commands

Lulu's Terminal User Interface is primarily driven by slash-commands (`/`).

### Core Commands
- `/help`: Show the help menu and available commands.
- `/provider`: Switch the active AI Provider. Shows a searchable list of available providers.
- `/model`: Switch the active AI Model. Fetches models directly from the active provider's API.
- `/dashboard`: Launch the local Web Dashboard for advanced settings and monitoring.
- `/edit`: Enter multi-line editing mode.
- `/clear`: Clear the current conversation history.
- `/exit` or `/quit`: Gracefully shutdown Lulu and close the terminal session.
