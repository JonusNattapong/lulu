# Welcome to Lulu 🤖

Lulu is an autonomous, open-source AI Assistant & Coding Agent designed to integrate directly into your development workflow. Born as an evolution of terminal-based agents, Lulu acts as a "Strong Brain" connecting you to a vast array of Large Language Models via multiple AI providers.

## Key Features

- ⚡ **Lightning Fast TUI:** Built with `ink`, featuring a highly responsive, keyboard-driven Terminal User Interface.
- 🧠 **Dynamic Provider & Model Selection:** Switch seamlessly between OpenAI, Anthropic, OpenRouter, Google, DeepSeek, Mistral, KiloCode, and more. Search models in real-time.
- 🔌 **Tool System:** Execute terminal commands, read/write files, and perform semantic search all autonomously.
- 🧩 **Agentic "Skill" Learning:** Lulu can learn and adapt to your project patterns over time.

To get started, simply run `bun run dev` or `npm run dev` in your terminal. Use the `/` command to explore available tools, such as `/provider` and `/model`.
