# Roadmap

The future of Lulu is focused on expanding its autonomy and integration capabilities.

- [x] **Dynamic Model Discovery:** Real-time fetching of models from provider APIs with resilient fallbacks.
- [x] **Unified TUI Search:** Seamlessly search models and commands inline.
- [ ] **Persistent Sessions:** Save and resume chat histories natively.
- [ ] **Local Embeddings (Transformers.js):** Zero-cost inference for local RAG and codebase understanding.
- [ ] **LSP Integration:** Connect directly to Neovim and VSCode Language Servers for native code awareness.
