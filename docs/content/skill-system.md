# Skill System

Lulu is not just a chatbot; it's an Agent. Skills are modular capabilities that can be dynamically loaded into the agent's context.

A skill consists of:
1. **Instructions:** A system prompt detailing *how* to perform the task.
2. **Tools:** A specific subset of tools (e.g., `run_command`, `replace_file_content`) required to execute the skill.

*(This system is currently under active development).*
